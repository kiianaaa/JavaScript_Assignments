Font

All fonts are from google fonts.

The way fonts were used is by importing the font link to css folder.
Then using the font in elements by using "font-family" and the name of the font.

Name:IBM Plex Mono used in:genra 
        @import url('https://fonts.googleapis.com/css2?family=IBM+Plex+Mono&display=swap');
        
        section h5 {
            color: rgb(22, 22, 22);
            cursor: context-menu;
            font-family: 'IBM Plex Mono', monospace;
        }
